## Dressage caller app

### problem
horse riders have to memorize their dressage test patterns and if they forget a part of the complex test it reduces their scores. 
they can use human callers that read the test outloud to them as they ride, but those are not always availble when practicing

### solution
an ios app that tracks the rider through the test and announces the next part of the test sequence through headphones or a speaker. 

### provides
- Audio cues for rider for tests
- reads Data public published for tests
- Need position data about rider in arena.
